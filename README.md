# cosh

Calculate the hyperbolic cosine of a number.

## Example

``` javascript
var cosh = require('cosh');

cosh(1);
// => 1.5430806348152437
```

## Installation

``` bash
$ npm install cosh
```

## API

``` javascript
var cosh = require('cosh');
```

### `cosh(x)`

Given a _Number_ `x`, returns the hyperbolic cosine of `x` as a _Number_.
